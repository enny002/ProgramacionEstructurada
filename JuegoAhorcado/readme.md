

1. el jugador tendra 5 turnos para adivinar la palabra (independientemente si en cada jugada tiene un acierto o un fallo)
en cada turno)

2. el jugador en cada turno vera lineas vacias y tendra que poner letras que el suponga que son las correctas de la palabra que tiene que encontrar
   
3. tendra 5 turnos para adivinar la palabra segun las letras que haya puesto
  
4. se debe preguntar al usuario si desea adivinar la palabra y de ser asi hacer la verificacion correspondiente
   

5. el juego se termina:
>> - el juego terminara cuando te acabes los 5 intentos de adivinar las palabra
>> - Cada dos intentos se te dara la opcion de adivinar la palabra entera. Y si no logras adivinar perderas de igual manera un intento.
